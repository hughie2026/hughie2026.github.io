# Personal Academic Website — Hughie Tao Xiaoyu

Static personal website built with plain HTML/CSS/JS, hosted on GitHub Pages.

## Local preview
Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Deploy on GitHub Pages
1. Create a repo named `<your-username>.github.io` (e.g. `hughietao.github.io`).
2. Push all files to the `main` branch.
3. In **Settings → Pages**, set Source to `main` / root.
4. Visit `https://<your-username>.github.io`.

## Update content
- Replace `assets/images/profile.jpg` with your own photo (recommended 600×600 px).
- Replace `assets/files/Xiaoyu_Tao_CV.pdf` with your latest CV.
- Edit each `.html` file to update text.
