# Personal Academic Website — Hughie Tao Xiaoyu

Static site (HTML/CSS/JS) for GitHub Pages.

## Pages
- `index.html` — Home (intro + full CV info)
- `academic.html` — Research, Publications, Teaching
- `contact.html` — Contact + embedded Taipa map

## Local preview
```bash
python3 -m http.server 8000
```
Visit http://localhost:8000

## Deploy
1. Create a repo named `<your-username>.github.io`.
2. Push all files to `main`.
3. Settings → Pages → Source: `main` / root.
